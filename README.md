# Jogo Campo Minado - Algoritmos e Programação (1º Semestre)

# 💣 Campo Minado em Java

Projeto desenvolvido em Java com o objetivo de recriar o clássico jogo Campo Minado, trabalhando conceitos fundamentais de lógica de programação, estruturas de repetição, condicionais e organização de código.

# 📌 Sobre o Projeto

Esse projeto foi criado como forma de praticar desenvolvimento em Java através de um jogo clássico.
Apesar de parecer simples, o Campo Minado exige bastante atenção na lógica por trás das verificações, posicionamento das bombas e interação do jogador com o tabuleiro.

Foi um projeto importante para desenvolver habilidades como:

- Planejamento antes da implementação
- Organização de classes e métodos
- Resolução de problemas
- Lógica condicional
- Manipulação de matrizes
- Tratamento de estados do jogo
